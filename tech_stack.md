# Our Tech Stack - The Tools We Love to Use

Here's what powers LeaveEase, our modern leave management system:

## **Backend & Foundation**
- **Java 17 with Spring Boot 3.5** - Our reliable workhorse for building robust, secure applications
- **Spring Security** - Keeping things safe with role-based access control for HR and employee panels
- **MongoDB** - Our flexible database that stores all leave requests, user data, and attendance records
- **JWT Tokens** - For secure user authentication and session management

## **Frontend Experience**
- **React 19** - Creating smooth, interactive user interfaces for both HR and employee dashboards
- **HTML, CSS & JavaScript** - The classic trio for polished, responsive design
- **WebSocket Support** - For real-time updates and interactive features

## **Communication & Workflows**
- **Spring Mail with SMTP** - Sending thoughtful email notifications through Gmail
- **RESTful APIs** - Making sure everything talks to each other seamlessly
- **WebFlux HTTP Client** - For smooth integrations with external services

## **Development & Deployment**
- **Maven** - Our trusty build tool for managing dependencies and packaging
- **Docker & Docker Compose** - Containerizing our app for consistent deployment
- **Spring Boot DevTools** - Making development faster with hot reload capabilities

## **What Makes It Special**
We've built LeaveEase with real-world HR teams in mind. Every technology choice was made to create a system that's:
- **Secure and reliable** for sensitive employee data
- **User-friendly** for both HR managers and employees
- **Scalable** to grow with your organization
- **Maintainable** with clean, well-structured code

This stack helps us deliver a leave management system that just works - no complicated AI, just solid engineering that solves real problems.
